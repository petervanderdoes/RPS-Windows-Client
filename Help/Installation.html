Installation

Since you're reading this, the installation instructions are really moot. Never the less, let me run through the details again in case you need to reinstall.

  1. Extract the contents of the RPS Digital View.zip file into a folder on a Windows drive
  2. That's all there is to it.

Initial Assumptions

The executable program makes certain assumptions when it is first installed.

  • The executable expects the database file (rps.mdb) to be in the same folder as the executable itself.
  • The images are expected to be stored in subfolders within the installation folder.

Both of these assumptions can be overridden with the File > Preferences... dialog.

Uninstall

Delete the executable (.exe) file and the database file (*.mdb)
